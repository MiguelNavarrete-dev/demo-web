# 🐾 Patitas Felices - Peluquería Canina (Demo)

¡Bienvenido al repositorio de **Patitas Felices**! Este es un proyecto de sitio web moderno y responsivo diseñado específicamente para una peluquería y spa canino en San Bernardo, Chile.

El objetivo de esta demo es demostrar habilidades de desarrollo frontend, creando una interfaz limpia (Soft UI), amigable y optimizada para la conversión de clientes.

## 🚀 Vista Previa
Puedes ver la demo en vivo aquí: [https://patitas-felices-hazel.vercel.app/]

## ✨ Características
* **Diseño Soft UI:** Uso de una paleta de colores pasteles (Blanco hueso, Celeste, Coral y Turquesa) para transmitir confianza y calma.
* **Layout Responsivo:** Totalmente adaptado para dispositivos móviles, tablets y escritorio.
* **Navbar Tipo Cápsula:** Navegación moderna con bordes redondeados y efecto flotante.
* **Modo Noche (Próximamente):** Estructura preparada para implementación de temas mediante variables CSS.
* **Integración con WhatsApp:** Botones estratégicos para agendamiento directo.
* **Componentes Modernos:** Footer con diseño asimétrico y botones de redes sociales flotantes.

## 🛠️ Tecnologías Usadas
* **HTML5** - Estructura semántica.
* **CSS3** - Flexbox, Grid, Variables CSS y Animaciones.
* **Bootstrap 5** - Sistema de grillas y componentes base.
* **JavaScript (Vanilla)** - Lógica de navegación y manipulación del DOM.
* **Google Maps API** - Para la sección de ubicación.
